# ModOS-Installer
This works by first downloading the DLLs' bytes from the repository _ModOS-DLLs_ 
then, it writes said bytes stright to new files on your PC (excluding EXEC.bin) 
then it gets the updater stuffs from the releases from _ModOS-DLLs_ 
Finally, it: Updates your calc

> **NOTE: ** EXEC.bin might get written to your PC before being written to the calc
